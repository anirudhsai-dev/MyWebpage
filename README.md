# Ganesh Personal Webpage

A polished static personal webpage for Ganesh, built with HTML and CSS and prepared for deployment on Vercel.

## Local preview

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import the repository into Vercel.
3. Choose the default static site settings.
4. Deploy — Vercel will serve `index.html` automatically.

The included `vercel.json` enables clean URLs and keeps the deployment configuration simple for this static site.
